4. Methodology & rates

Rates aligned with the published reference site & cross-checked against Azure retail pricing.

Volume model

  • Light: 5 messages / user / day
  • Typical: 20 messages / user / day
  • Heavy: 60 messages / user / day
  • 22 working days / month
  • 1 message ≈ 1 text record (Comm Compliance simplification)

Per-meter unit math

  • Communication Compliance: (msgs / 1,000) × $0.50
  • Data Lifecycle Mgmt: (msgs / 1,000,000) × $6.00
  • Insider Risk Mgmt: (msgs × 0.10 / 10,000) × $25.00 — ~10% of msgs trigger an activity
  • Audit (3rd party): (msgs × 1.5 / 1,000,000) × $15.00
  • Collection Policies (3rd party): (msgs × 2.5 / 10,000) × $0.50
  • DSI: (GB × $5.00) + (Units × $5.00)
  • eDiscovery Premium: GB × $20.00
